
Website Structure:

1. index.html: This file contains the main homepage of the website, including the header, navigation menu, welcome section, project section, audio controls, pricing section, info list, service list, and footer. It also includes a youtube link, a personal video, a personal audio, and free stock images

2. services.html: This file contains the services page of the website, including the header, navigation menu, pricing section, info list, service list, and footer.

3. contact.html: This file contains the contact page of the website, including the header, navigation menu, contact section, and footer.

4. styles.css: This file contains the CSS stylesheet for styling the HTML elements across all web pages.

5. Assignment2 folder: This folder contains the multimedia files (images, videos, audio) used in the website, referenced by the HTML files.

Function Documentation:

1. Header Section Styling:
   - Purpose: To style the header section of the website for consistent branding and visual appeal.
   - Dependent Code Structures: HTML `<header>` element, CSS styling for background color, text color, padding, and text alignment.
   - Example: `<header><h1>Welcome to Nathan's Personal Website</h1></header>`

2. Project Section Styling:
   - Purpose: To style the project section of the website for showcasing featured projects in a visually appealing manner.
   - Dependent Code Structures: HTML `<section>` element, CSS styling for flexbox layout, image size, border-radius, and text alignment.
   - Example: `<section><div><img src="project.jpg" alt="Project"><h3>Project Title</h3><p>Project Description</p></div></section>`

3. Audio Controls Styling:
   - Purpose: To style the audio controls section of the website for easy playback and control of audio content.
   - Dependent Code Structures: HTML `<audio>` element, CSS styling for customizing the appearance and layout of audio controls.
   - Example: `<audio controls><source src="audio.mp3" type="audio/mpeg">Your browser does not support the audio element.</audio>`

4. Pricing Section and Tables Styling:
   - Purpose: To style the pricing section and tables of the website for clear presentation of pricing information.
   - Dependent Code Structures: HTML `<section>` and `<table>` elements, CSS styling for table layout, headers, and data cells.
   - Example: `<section><table><tr><th>Package</th><th>Description</th><th>Price</th></tr><tr><td>Basic</td><td>Includes basic features</td><td>$99</td></tr></table></section>`

5. Info List and Service List Styling:
   - Purpose: To style the info list and service list sections of the website for presenting information in a concise and visually appealing manner.
   - Dependent Code Structures: HTML `<ul>` and `<li>` elements, CSS styling for list layout, spacing, and bullet points.
   - Example: `<ul><li>Info 1</li><li>Info 2</li><li>Info 3</li></ul>`

6. Footer Styling:
   - Purpose: To style the footer section of the website for providing contact information and navigation links.
   - Dependent Code Structures: HTML `<footer>` element, CSS styling for text alignment, background color, padding, and text color.
   - Example: `<footer><p>&copy; Nathan's Website.</p></footer>`
